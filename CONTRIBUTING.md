# Contributing to ZKCG Verifier

Thanks for your interest in contributing!

This repository contains the public verifier and protocol code. Contributions here help make verification more robust, usable, and well-documented.

### Guidelines

- Please open an **issue** first to discuss major changes or new features.
- Focus areas we especially welcome:
  - Bug fixes in verification logic
  - Documentation improvements
  - Example code and tests
  - Performance optimizations
  - Better error messages and API ergonomics

All contributions are licensed under **Apache-2.0**.
